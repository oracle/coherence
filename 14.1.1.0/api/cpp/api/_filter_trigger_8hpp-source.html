00025 /**
00026 * A generic Filter-based MapTrigger implementation. If an evaluation of the
00027 * coherence::util::MapTrigger::Entry object representing a pending
00028 * change fails (returns false), then one of the following actions is taken:
00029 * <ul>
00030 *   <li> action_rollback - an IllegalArgumentException is thrown by the
00031 *        trigger to reject the operation that would result in this change
00032 *        (default);
00033 *   <li> action_ignore - the change is ignored and the Entry's value is reset
00034 *        to the original value returned by the
00035 *        coherence::util::MapTrigger::Entry#getOriginalValue() method;
00036 *   <li> action_remove - the entry is removed from the underlying backing map
00037 *        using the coherence::util::InvocableMap::Entry#remove(bool) call.
00038 * </ul>
00039 *
00040 * Note: This trigger never prevents entries from being removed.
00041 *
00042 * @author djl  2008.05.22
00043 */
